Installation
------------

To use nanoaod-base-analysis, first install it:

.. code-block:: console
    
    git clone https://gitlab.cern.ch/cms-phys-ciemat/nanoaod_base_analysis.git -b py3 nanoaod_base_analysis
    cd nanoaod_base_analysis
    source setup.sh
    law index --verbose
